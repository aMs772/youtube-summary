# Youtube Summary

This project summarises a youtube video and provides a short quiz of 5 questions on that video using openai API key.

# Installation
```
git clone <githublink>
cd youtube-summary
uv sync
```

# Requirements
# 1) openai API key:
Go to openai website: https://openai.com/api/  
login with your account and create a key and save it  
create a .env file in this directory and store your key as  
```
OPENAI_API_KEY=sk.........
```

# Usage
After following installation and requirements, you can run main.py. If everything went right a browser window will open asking for access to google docs. After giving access a google doc link will be printed on your terminal.  
The final doc will be saved in your drive by the name of YouTube Knowledge Pipeline.
